Where does the agent runtime live? / Inside Electron or its own process.

Sidecar Daemon

A separate dycoded Node process owns the agents, SQLite, and orchestration. Electron is a thin client over WebSocket. Close the IDE, agents keep working. Headless mode is a free side-effect — connect from another machine, or no UI at all.

Crash-resilient Headless capable Multi-client
BEST FORLong-running squads, overnight runs, server deployment (multica-style)
