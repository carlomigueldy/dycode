How is the fleet organized? / Stable groups, role crews, or hybrid.

The Control Room renders agents under some structure. This decision determines what the fleet view groups by, how work routes to a worker, and whether the user pre-defines groups or the system assembles them per-task. It also dictates which shadcn primitives carry the load — Sidebar with collapsible groups, Command for crew composition, or both.
